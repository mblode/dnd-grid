## Exports

Common exports include:

```tsx
import {
  DndGrid,
  GridItem,
  useContainerWidth,
  useDndGridResponsiveLayout,
  useEdgeScroll,
  AutoScrollActivator,
  TraversalOrder,
  verticalCompactor,
  defaultConstraints,
  type AutoScrollOptions,
  type Compactor,
  type LayoutConstraint,
  type Layout,
  type LayoutItem,
  type ResponsiveLayouts,
} from "@dnd-grid/react";
```

## Components

| Component | Description |
|-----------|-------------|
| [`DndGrid`](/api-reference/dnd-grid) | Main grid layout component |
| `GridItem` | Individual grid item (used internally) |

## Hooks

| Hook | Description |
|------|-------------|
| `useContainerWidth` | Measure container width with ResizeObserver |
| `useDndGridResponsiveLayout` | Manage responsive breakpoints + layout state |
| `useEdgeScroll` | Auto-scroll while dragging near scroll edges |

## Types

| Type | Description |
|------|-------------|
| [`Layout`](/api-reference/layout-item#layout) | Array of layout items (`LayoutItem[]`) |
| [`LayoutItem`](/api-reference/layout-item) | Position and size of a single grid item |
| `ResponsiveLayouts` | Breakpoint â†’ layout map |
| `Compactor` | Pluggable compaction strategy |
| `LayoutConstraint` | Drag/resize constraint definition |
| `AutoScrollOptions` | Configuration for edge auto-scrolling |
| `AutoScrollActivator` | Choose pointer or draggable-rect activation |
| `TraversalOrder` | Control scroll ancestor traversal order |

## CSS Classes

The grid applies these CSS classes that you can style:

| Class | Element |
|-------|---------|
| `.dnd-grid` | Grid container |
| `.dnd-grid-item` | Each grid item |
| `.dnd-grid-item-content` | Non-placeholder grid items |
| `.dnd-grid-placeholder` | Placeholder during drag/resize |
| `.dnd-draggable` | Draggable items |
| `.dnd-draggable-dragging` | Item currently being dragged |
| `.resizing` | Item currently being resized |
| `.static` | Static (locked) items |
| `.dnd-grid-animating` | Item settling after drag |
| `.dropping` | Dropping placeholder item |
| `.dnd-grid-resize-handle` | Resize handle element |
| `body.dnd-grid-dragging` | Global dragging state |
| `body.dnd-grid-resizing` | Global resizing state |

## Quick Reference

### Essential Props

```tsx
<DndGrid
  // Required
  layout={layout}           // Array of LayoutItem
  width={1200}              // Container width in pixels

  // Grid configuration
  cols={12}                 // Number of columns
  rowHeight={150}           // Row height in pixels
  margin={10}               // Gap between items (number or { top, right, bottom, left })

  // Behavior
  isDraggable={true}        // Enable dragging
  isResizable={true}        // Enable resizing
  autoScroll={true}         // Edge auto-scroll while dragging
  isBounded={false}         // Keep items in bounds
  compactor={verticalCompactor} // Compaction strategy
  constraints={defaultConstraints} // Drag/resize constraints

  // Callbacks
  onLayoutChange={(layout) => {}}
  onDragStart={(layout, oldItem, newItem, placeholder, e, node) => {}}
  onDrag={(layout, oldItem, newItem, placeholder, e, node) => {}}
  onDragStop={(layout, oldItem, newItem, placeholder, e, node) => {}}
>
  {children}
</DndGrid>
```

### LayoutItem Structure

```tsx
interface LayoutItem {
  i: string;           // Unique identifier
  x: number;           // X position (grid units)
  y: number;           // Y position (grid units)
  w: number;           // Width (grid units)
  h: number;           // Height (grid units)
  minW?: number;       // Min width
  maxW?: number;       // Max width
  minH?: number;       // Min height
  maxH?: number;       // Max height
  static?: boolean;    // Non-movable
  isDraggable?: boolean;
  isResizable?: boolean;
  isBounded?: boolean;
}
```
