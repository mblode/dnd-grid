# dnd-grid

A powerful, flexible grid layout system for React with drag-and-drop and resize capabilities.

## Features

<CardGroup cols={2}>
  <Card title="100% React" icon="react">
    No jQuery or external dependencies. Pure React implementation with full TypeScript support.
  </Card>
  <Card title="Draggable & Resizable" icon="arrows-up-down-left-right">
    Drag items to reposition and resize from any edge or corner with configurable handles.
  </Card>
  <Card title="Responsive Breakpoints" icon="display">
    Define different layouts for different screen sizes with automatic breakpoint switching.
  </Card>
  <Card title="Server-Side Ready" icon="server">
    Compatible with server-rendered applications including Next.js and Remix.
  </Card>
</CardGroup>

## Why dnd-grid?

Unlike other grid systems, dnd-grid is designed specifically for React with modern patterns:

- **CSS Transforms** - Smooth animations using GPU-accelerated transforms
- **Configurable Compaction** - Vertical, horizontal, or no compaction
- **Collision Detection** - Intelligent item pushing or overlap prevention
- **Static Items** - Lock specific items in place
- **Layout Persistence** - Serialize and restore layouts easily

## Quick Example

```tsx
import { DndGrid, type Layout } from "@dnd-grid/react";
import { useState } from "react";

const initialLayout: Layout = [
  { i: "a", x: 0, y: 0, w: 2, h: 2, deg: 0 },
  { i: "b", x: 2, y: 0, w: 2, h: 2, deg: 0 },
  { i: "c", x: 4, y: 0, w: 2, h: 2, deg: 0 },
];

function MyGrid() {
  const [layout, setLayout] = useState(initialLayout);

  return (
    <DndGrid
      layout={layout}
      cols={12}
      rowHeight={30}
      width={1200}
      onLayoutChange={setLayout}
    >
      <div key="a">A</div>
      <div key="b">B</div>
      <div key="c">C</div>
    </DndGrid>
  );
}
```
