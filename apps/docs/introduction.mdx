# dnd-grid

A lightweight, performant grid layout for React. Drag, resize, and auto-scale with zero external dependencies.

## Usage

```tsx
import { DndGrid } from "@dnd-grid/react";
import "@dnd-grid/react/styles.css";

export default function Dashboard() {
  return (
    <DndGrid
      cols={12}
      rowHeight={50}
      width={1200}
      layout={[
        { i: "a", x: 0, y: 0, w: 4, h: 2 },
        { i: "b", x: 4, y: 0, w: 4, h: 2 },
      ]}
    >
      <div key="a">Item A</div>
      <div key="b">Item B</div>
    </DndGrid>
  );
}
```

## Features

- **Performance:** GPU-accelerated CSS transforms and minimal re-renders.
- **Zero Dependencies:** Pure React implementation. No jQuery.
- **Reliability:** First-class TypeScript support with strict definitions.
- **Flexibility:** Custom compaction modes (vertical/horizontal) and collision handling.
- **SSR Ready:** Compatible with Next.js, Remix, and other server-side frameworks.
