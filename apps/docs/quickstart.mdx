## Basic example

```tsx
import { useState } from "react";
import { DndGrid, type Layout } from "@dnd-grid/react";
import "@dnd-grid/react/styles.css";

const initialLayout: Layout = [
  { i: "a", x: 0, y: 0, w: 2, h: 2 },
  { i: "b", x: 2, y: 0, w: 2, h: 2 },
];

export default function App() {
  const [layout, setLayout] = useState(initialLayout);

  return (
    <DndGrid
      layout={layout}
      onLayoutChange={setLayout}
      cols={12}
      rowHeight={40}
      width={800}
    >
      <div key="a" style={{ background: "#eee" }}>A</div>
      <div key="b" style={{ background: "#eee" }}>B</div>
    </DndGrid>
  );
}
```

## How it works

`dnd-grid` is controlled. The `layout` prop is the source of truth.

1. **Layout state:** An array of items with `x`, `y`, `w`, and `h`.
2. **Matching keys:** Each layout `i` must match its child `key`.
3. **Grid units:** Sizes are in columns and rows, not pixels.
