## Basic Example

```tsx
import { useState } from "react";
import { DndGrid, type Layout } from "@dnd-grid/react";
import "@dnd-grid/react/styles.css";

// 1. Define the initial state
const initialLayout: Layout = [
  { i: "a", x: 0, y: 0, w: 2, h: 2 },
  { i: "b", x: 2, y: 0, w: 2, h: 2 },
];

export default function App() {
  const [layout, setLayout] = useState(initialLayout);

  return (
    <DndGrid
      layout={layout}
      onLayoutChange={setLayout}
      cols={12}
      rowHeight={40}
      width={800}
    >
      <div key="a" style={{ background: "#eee" }}>A</div>
      <div key="b" style={{ background: "#eee" }}>B</div>
    </DndGrid>
  );
}
```

## The Mental Model

`dnd-grid` is **controlled**. The `layout` prop is the single source of truth.

1.  **Layout State:** An array of objects defining position (`x`, `y`) and size (`w`, `h`).
2.  **Unique Keys:** The `i` property in the layout **must** match the child's `key`.
3.  **Grid Units:** All dimensions are in columns/rows, not pixels.
