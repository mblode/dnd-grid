## Set up

Install the package and import the required styles.

<CodeGroup>

```bash npm
npm install @dnd-grid/react
```

```bash yarn
yarn add @dnd-grid/react
```

```bash pnpm
pnpm add @dnd-grid/react
```

</CodeGroup>

```tsx
import "@dnd-grid/react/styles.css";
```
