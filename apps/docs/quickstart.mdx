## Installation

Install dnd-grid using your preferred package manager:

<CodeGroup>

```bash npm
npm install @dnd-grid/react
```

```bash yarn
yarn add @dnd-grid/react
```

```bash pnpm
pnpm add @dnd-grid/react
```

</CodeGroup>

## Import Styles

Import the required CSS styles in your application:

```tsx
import "@dnd-grid/react/dist/styles.css";
```

Or in your CSS file:

```css
@import "@dnd-grid/react/dist/styles.css";
```

## Basic Usage

Here's a minimal example to get you started:

```tsx
import { useState } from "react";
import { DndGrid, type Layout } from "@dnd-grid/react";
import "@dnd-grid/react/dist/styles.css";

const initialLayout: Layout = [
  { i: "a", x: 0, y: 0, w: 2, h: 2, deg: 0 },
  { i: "b", x: 2, y: 0, w: 2, h: 2, deg: 0 },
  { i: "c", x: 4, y: 0, w: 2, h: 2, deg: 0 },
];

function App() {
  const [layout, setLayout] = useState(initialLayout);

  return (
    <DndGrid
      layout={layout}
      cols={12}
      rowHeight={30}
      width={1200}
      onLayoutChange={(newLayout) => setLayout(newLayout)}
    >
      <div key="a">Item A</div>
      <div key="b">Item B</div>
      <div key="c">Item C</div>
    </DndGrid>
  );
}
```

<Note>
  Each child element must have a unique `key` prop that matches the `i` property in the layout item.
</Note>

## Understanding the Layout

The layout is an array of objects describing each grid item's position and size:

```tsx
interface LayoutItem {
  i: string;    // Unique identifier (must match child key)
  x: number;    // X position in grid units (0-based)
  y: number;    // Y position in grid units (0-based)
  w: number;    // Width in grid units
  h: number;    // Height in grid units
  deg?: number; // Rotation in degrees (optional)
}
```

## Grid Dimensions

The grid is divided into columns. Each item's `x`, `y`, `w`, and `h` are expressed in grid units:

- **cols** - Total number of columns (default: 12)
- **rowHeight** - Height of each row in pixels (default: 150)
- **width** - Total width of the container in pixels (required)
