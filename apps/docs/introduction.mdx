# dnd-grid

A drag-and-drop (DnD), resizable grid layout for React with responsive hooks and styling controls.

## Usage

```tsx
import { DndGrid } from "@dnd-grid/react";
import "@dnd-grid/react/styles.css";

export default function Dashboard() {
  return (
    <DndGrid
      cols={12}
      rowHeight={50}
      width={1200}
      layout={[
        { i: "a", x: 0, y: 0, w: 4, h: 2 },
        { i: "b", x: 4, y: 0, w: 4, h: 2 },
      ]}
    >
      <div key="a">Item A</div>
      <div key="b">Item B</div>
    </DndGrid>
  );
}
```

## Features

- **Drag + resize:** Configurable handles and smooth interactions.
- **Responsive layouts:** Hooks for breakpoints and container width.
- **Compaction + constraints:** Control packing and bounds behavior.
- **TypeScript + SSR:** Works with modern React and server-rendered apps.
- **Styling hooks:** CSS variables, data attributes, and item state hook.
